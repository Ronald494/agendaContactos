**AgendaContactos**

AgendaContactos desarrollada en C# que permite registrar, buscar, editar y eliminar contactos personales.
El objetivo del proyecto es practicar la gestión de datos y el uso de listas en C#, además de aplicar control de versiones con Git.

**Características principales**

- Agregar nuevos contactos (nombre, teléfono, correo) 

- Buscar contactos por nombre

- Editar la información de un contacto existente

- Eliminar contactos del registro

- Mostrar la lista completa de contactos

- Guardar los datos en un archivo para mantener la información
